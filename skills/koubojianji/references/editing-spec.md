# 口播剪辑与品牌包装规范

## 1. 基础剪辑

- 使用 9:16、1080×1920 竖屏构图。人物主镜头保持稳定，避免持续晃动、旋转和频繁推拉。
- 用自然跳剪处理口误和停顿；必要时使用约 100%–106% 的轻微构图变化遮盖剪切点。
- 画面、字幕、卡片和画中画分别设置安全区域。不得遮挡人脸、关键按钮、正文或抖音右侧交互区。
- 调色以自然肤色、清晰面部和受控高光为目标，避免过度磨皮和明显滤镜。

## 2. 双语字幕

- 中文在上、英文在下。中文为主字幕，英文缩小一档。
- 按自然短语切分，每组只表达一个阅读单元；一组字幕完整退出后才能显示下一组。
- 英文采用适合短视频阅读的简洁意译，保留原意和专有名词，不增加原文没有的判断。
- 普通字幕使用轻微上移和淡入；只有重点标题从下方弹出。
- 字幕保持底部居中，但应高于平台底部控件，并与圆形画中画保留明确间距。
- 输出双语、中文、英文三套 SRT，并检查时间码、断句、拼写和标点。

## 3. 重点信息卡

适用于每条 30–60 秒视频中的 2–3 个关键结论、步骤或工具信息。

- 外观：抽象手机界面结构，可含状态栏、返回按钮、搜索框和播放按钮，但不得出现真实 App 名称、Logo 或水印。
- 背景：深棕黑、深灰或深紫黑，带轻微磨砂质感。
- 卡片：半透明玻璃拟态、圆角、深蓝紫底色、轻微粉紫霓虹边框。
- 装饰：缓慢流动的霓虹线、圆弧、微弱光点；装饰不得干扰正文。
- 文字：粗体中文无衬线；主标题白色；重点词使用粉紫渐变。
- 步骤卡按内容自动显示 1–3 张，不强行凑成三个步骤。
- 动效参考：卡片进入 0.35–0.55 秒；步骤间错峰 0.12–0.18 秒；上下浮动 4–8 像素、周期 3–5 秒；霓虹线流动周期 6–10 秒。禁止无限循环，按场景时长计算有限重复次数。

## 4. 软件演示与圆形画中画

- 软件、网页、图片或视频内容占据主画面。围绕当前操作区域裁切、平移和轻微缩放，不得把整个横屏桌面压缩成无法阅读的小字。
- 口播人物必须使用圆形蒙版，默认直径为画面宽度的 21%，允许在 18%–24% 间调整。
- 默认放左下角；若遮挡字幕、界面按钮或主要操作区，则放右下角。
- 使用轻微白色描边或柔和阴影分离人物与背景，不得改成方形或圆角矩形。
- 人物全屏切换到画中画时，以 0.45–0.65 秒自然缩小并移动；返回人物全屏时反向恢复。
- 允许添加鼠标点击反馈、高亮框、虚线框和箭头。单一时刻只强调一个操作重点。
- 信息层级固定为：主界面 > 关键字幕 > 圆形口播画中画。

## 5. 品牌系统

唯一品牌资产为 `assets/xingxingdaren-logo.svg`，不得主动从网站抓取其他元素。

- 品牌绿 `#11A683`：成功、完成、推荐状态。
- 品牌红 `#E6362F`：警告、错误或极少量关键提醒。
- 品牌紫 `#8A74E8`：主要科技霓虹色。
- 深色 `#171A1F`：背景和正文深色。
- 米白 `#F8F9F7`：浅色背景、描边和高对比文字。
- 重点 UI 以品牌紫为主；绿色和红色只作为状态色，避免三种强调色同时争夺注意力。
- 转场可拆解双 X 斜线、电路线和节点，做短促交叉划入、线路延伸或节点点亮。
- 结尾页使用深色背景、完整 Logo 和简短行动引导。线路与节点缓慢点亮，Logo 轻微放大进入。
- 首条正式视频应建立 `DESIGN.md`，记录最终字体、人物安全区、字幕尺寸、配色和禁止事项；后续视频复用。

## 6. 声音

- 轻度降噪，不得把人声处理得发闷或产生水下感。
- 适度均衡、压缩和响度统一，目标约为 `-16 LUFS`，峰值不超过 `-1 dB`。
- 背景音乐保持低音量；只有重点标题、界面切换或操作确认可使用极少量音效。

## 7. 导出与验收

默认导出：

- MP4、H.264、1080×1920、30fps、约 12 Mbps。
- AAC、48kHz。

交付前必须：

1. 通过 HyperFrames `lint`、`validate` 和 `inspect`。
2. 新建或大幅修改动画时检查动画映射中的碰撞、不可见元素、越界和不合理节奏。
3. 完整观看成片，确认没有黑帧、闪帧、跳音、音画不同步或错误裁切。
4. 检查中英文字幕同步、翻译、断句、拼写、可读性和退出状态。
5. 检查 UI 无乱码、无真实平台 Logo、无账号隐私、无水印。
6. 检查圆形画中画、双语字幕、主界面和平台控件互不遮挡。
7. 检查音乐不压人声，重点音效不过量。
